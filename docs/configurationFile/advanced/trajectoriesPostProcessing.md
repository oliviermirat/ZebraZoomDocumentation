# Trajectories post-processing

## Head position over time

Post-processing available by adding in the configuration file:

```
 'postProcessHeadingWithTrajectoryAdvanced': 1
```

## Heading values over time

Post-processing available by adding in the configuration file:

```
 'postProcessMultipleTrajectories': 1,
 'postProcessMaxDisapearanceFrames': 10,
 'postProcessMaxDistanceAuthorized': 152,
```
